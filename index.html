How it works

Keystroke dynamics analyzes typing characteristics of the user and creates an individual biometric profile. Based on the unique typing patterns KeystrokeDNA verifies each login attempt and ensures that the identity is not forged.

KeystrokeDNA can be used as a primary or secondary mean of authentication, fraud prevention, and for retrospective identity analysis. It can be conveniently integrated in any web application with just a few lines of code.

HOW you type, not WHAT you type

Keystroke dynamics monitors the way person interacts with the keyboard by analyzing typing manner, regardless language, words or symbols.

Effective on keyboard and keypad

KeystrokeDNA can recognize the identity of the user typing on both keyboard and touchscreen as the typing pattern remains the same.

Simple integration

KeystrokeDNA can be easily integrated into any web application with a few lines of code. It does not require specific knowledge or efforts.

Resistant to password attacks

Keystroke dynamics is a strong method of authentication that recognizes a user. Even if the password was hacked the access will be denied.

More reliable than biometric

Unlike static biometric authentication methods keystroke dynamics recognizes the behavioral pattern that cannot be recorded or copied.

Does not require special hardware

KeystrokeDNA requires only a keyboard (keypad) that makes it the most convenient and affordable method of biometric authentication.

Identity confirmation

KeystrokeDNA recognizes the identity and verifies the user based on his typing characteristics. KeystrokeDNA collects the data on how the user interacts with the keyboard or keypad and creates a unique behavioral biometric profile that is used as a template for authentication.

User verification

Credential-sharing is an essential problem for the non-biometric authentication. It is difficult or sometimes even impossible to distinguish the forgery. KeystrokeDNA solves the problem by verifying user’s identity with the 99% accuracy using a personal typing manner that cannot be copied.

Convenient authorization

KeystrokeDNA ensures fast and convenient authorization. The user is not required to take any additional actions to confirm the identity as the verification based on the person’s unique typing characteristics is done automatically.

Data security

KeystrokeDNA provides high level of data security, detects fraud and prevents unauthorized access.

Secure authentication

With the keystroke dynamics as a secondary authentication method the consequences of stolen credentials are less harmful. KeystrokeDNA checks the legitimacy of every login attempt correlating it with the user’s unique typing pattern collected before.

Fraud prevention

KeystrokeDNA enables an effective data protection from any potential fraud. No one can forge user’s identity to access confidential data. KeystrokeDNA denies the access if the typing rhythm does not match.
